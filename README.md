# Camila Bento — Psicóloga

Premium landing page for a clinical psychologist based in Ciudad del Este, Paraguay. Neural Depth design concept: a Spline 3D interactive brain as the hero centerpiece, a custom neural cursor trail, magnetic buttons, and a seamless dark/light mode toggle. Built to communicate trust, depth, and modernity in a single scroll.

## Features

- Spline 3D interactive brain embedded as full-screen hero
- Custom neural cursor trail animation
- Magnetic button interactions on key CTAs
- Dark/light mode toggle with smooth transitions
- Testimonials slider with complex state management
- WhatsApp CTA for direct appointment booking
- Fully responsive, mobile-first layout
- No frameworks — pure HTML, CSS, and JavaScript

## Tech Stack

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 (custom properties + dark/light theming) |
| Logic | Vanilla JavaScript (ES6+) |
| 3D Hero | Spline (spline.design) |
| Images | WebP optimized |
| Hosting | Cloudflare Pages |
